02 · show, don't tell

Real Rust, compiled to WebAssembly

wasm/lab.rs → rustc --target wasm32-unknown-unknown →lab.wasm(843 B, loaded lazily). Each function is a slice of an actual project. The numbers below are computed in WebAssembly, not JavaScript.

// QuorumRAG.rs — Reciprocal Rank Fusion (the scoring core)
pub fn rrf_term(k: i32, rank: i32) -> f64 {
    if rank <= 0 { 0.0 } else { 1.0 / ((k + rank) as f64) }
}

// Kademlia DHT — XOR distance metric + k-bucket index
pub fn xor_distance(a: u32, b: u32) -> u32 { a ^ b }
pub fn bucket_index(a: u32, b: u32) -> u32 {
    (a ^ b).leading_zeros()
}

// Content addressing — FNV-1a over a host-written buffer
pub fn fnv1a32(bytes: &[u8]) -> u32 {
    let mut h = 0x811c_9dc5u32;
    for &x in bytes { h ^= x as u32; h = h.wrapping_mul(0x0100_0193); }
    h
}

Colophon

Built with Astro and hand-written CSS — no UI framework, no Tailwind. Type isManrope, Cormorant Garamond, and JetBrains Mono, self-hosted. Deployed as static files to GitHub Pages; the only server anywhere is one Cloudflare Worker proxying Spotify.

The WebAssembly on this page is genuinely rustc-compiled fromwasm/lab.rs — 843 bytes, no toolchain magic. This page sets no cookies; it useslocalStorage only to remember your theme, mute, and terminal history.
